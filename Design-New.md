# Šimon Havlík Portfolio - New Design

## Design System

### Colors
- Primary Background: #FFFFFF (white)
- Secondary Background: #F8F9FA (light gray)
- Text Primary: #1A1A1A (near black)
- Text Secondary: #6B7280 (gray)
- Accent: #DC2626 (red for China section)
- Accent Blue: #3B82F6 (for programming)
- Accent Green: #10B981 (for finance)
- Accent Purple: #8B5CF6 (for design)
- Accent Orange: #F59E0B (for languages)

### Typography
- Headings: Playfair Display (serif) - elegant, professional
- Body: Inter (sans-serif) - clean, readable
- Mono: JetBrains Mono for code/technical text

### Layout
- Max width: 1200px
- Padding: 80px vertical sections
- Clean, minimal, lots of whitespace
